# Bagel Backend Project - Claude Development Guide

## 프로젝트 개요
Spring Boot 기반의 백엔드 API 서버 프로젝트입니다.

## 기술 스택
- **Language**: Java 17
- **Framework**: Spring Boot 3.4.1
- **Build Tool**: Gradle
- **Database**: PostgreSQL
- **Authentication**: JWT (jjwt 0.12.3)
- **API Documentation**: Springdoc OpenAPI (Swagger) 2.3.0
